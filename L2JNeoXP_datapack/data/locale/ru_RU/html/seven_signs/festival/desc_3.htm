Руководство по фестивалю:
[ Фестиваль Тьмы - Требования к участию ]
1. Для участия в Фестивале Тьмы вы должны быть членом либо Лордов Рассвета, либо Революционеров Заката.
2. Вы можете участвовать в любое время в течение периода соревнований Семи Знаков, и нет ограничения на количество участий.
3. Для запроса участия необходимо, чтобы пять или более игроков, принадлежащих к одной группе бойцов, сформировали группу.
4. Уровни участников группы должны соответствовать критериям, установленным фестивалем. Вот список уровней, используемых в Фестивале Тьмы:
- Уровень ниже 32-й категории - Уровень ниже 43-й категории - Уровень ниже 54-й категории - Уровень ниже 65-й категории - Без ограничения уровня
5. Уровни всех участников группы должны быть одинаковыми.
