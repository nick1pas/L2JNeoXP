Функции КланХолла активированы.
Плата за использование каждой функции будет автоматически списываться со счета склада клана за каждый период времени. Если на счету больше нет средств, соответствующая функция будет сброшена.
Чтобы проверить оставшееся время, доступное для каждой функции, воспользуйтесь меню управления кланхоллом.
