Функции замка активированы.
Плата за использование каждой функции будет автоматически вычитаться из счета клановой казны для каждого периода времени. Если средств на счете больше нет, соответствующая функция будет сброшена.
Чтобы проверить оставшееся время для каждой функции, используйте меню Управления Замком.
